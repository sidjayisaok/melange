Melange: Web tools + d3.js


Melange works in conjunction with the d3.js library.
The d3.js library can render both simple and complex interactive and static graphics using scalable vector graphics (SVG).
Tools used include a pie graph, bar chart, and bullet chart. Data comes from dynamically generated sources or csv files.
